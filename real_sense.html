Intel RealSense Puppet

Whilst on my placement at Rhythm Digital I got to play around with Intel's new RealSense cameras.

These cameras function like a Kinect, using infrared technology to measure depth as well as colour images. Using the camera I created a real time puppet Windows App. The user can control the limbs of the puppet using thier fingertips and the face of the puppet is a 3D image of their face.

This app was made using the Cinder OpenGL api as well as Box2D for the physics engine and Intel's RealSense SDK for camera control.
