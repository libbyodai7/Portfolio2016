Keyboard Alphabet

Produced in collaboration with Luke Monet, the alphabet keyboard is an interactive web experience which offers a new way to think about typography.

We made this in the first year of university. I was responsible for the coding whilst Luke handled the design. We produced this as part of an alphabet photography project where we took photographs of letters hidden in everyday items and forms. Press the keyboard keys to see our photographs. There is also a hidden song in the project, see if you can find it!
